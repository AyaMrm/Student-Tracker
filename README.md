## Overview

**Grade** is an academic management platform designed primarily for **universities**.  
It is built entirely using **Java and JavaFX**, with a focus on a **simple, modern, and user-friendly design**.  
The platform centralizes academic data and workflows for students, teachers, and administrators.

---

## Technologies Used

- **Java**
- **JavaFX**
- Desktop-based graphical user interface
- Modular and role-based architecture

---

## User Roles

The platform supports **three types of users**:

- Student
- Professor
- Administrator

Each role has dedicated features and access levels.

---

## Student Features

Students can log in to the platform to access all their academic information, including:

- View grades and exam results
- Check class schedules
- See their **ranking within the class**
- Access personal and academic information
- View transcripts from previous academic years
- Access enrolled courses and assigned projects
- See assigned teachers for each course

This allows students to have a **complete academic overview in one place**.

---

## Professor Features

Professors can manage academic sessions and student performance by:

- Viewing student lists for each scheduled session
- Recording student attendance
- Submitting exam and test grades
- Uploading course materials
- Assigning homework and academic content

These features help streamline teaching activities and academic tracking.

---

## Administrator Features

Administrators have full control over the platform and user management:

- Search for students, professors, and other administrators
- Add new users
- Modify existing user information
- Delete user accounts
- Maintain system consistency and data integrity

---

## Design Philosophy

- Simple and clean interface
- Modern UI using JavaFX components
- Role-based access control
- Easy navigation for all user types

---

## Conclusion

Grade provides a **centralized and efficient academic management solution** for universities.  
By combining a modern interface with clear role separation, the platform simplifies academic monitoring, communication, and administration.
